### 面试题16：反转链表

#### 题目：
定义一个函数，输入一个链表的头结点，反转该链表并输出反转后链表的头结点。<br/>
链表的结点定义如下：
```
class ListNode{
	int m_nvalue;
	ListNode m_pNext;
}
```


#### 解题思路：
解题思路1（不太好）：将链表依次放入栈中，然后再将其弹出，将其连接到下一个弹出的结点
<br/>
<br/>
解题思路2（推荐）：直接遍历链表一遍，将当前遍历的结点，指向上一个结点（之所以要指向上一个结点，不是下个结点指向当前结点，是因为下个结点要留着做遍历），直到当前结点为null，需要使用一个临时结点存储上一个结点，不然会操成链表断裂<br/>
