### 面试题12：打印1到最大的n位数
#### 题目：
输入数字n，按顺序打印出从1最大的n位十进制数。比如输入3，则打印出1、2、3一直到最大的3位数即999.<br/>
#### 解题思路
解题思路1：因为要考虑大数问题，将数字化为字符串，每次在字符串的基础上加1，然后在判断是否是最大数的时候，通过最左边的位有没有进位来判断
<br/>
解题思路2：利把数字的每一位都从0到9排序一遍，就可以得到所有的十进制数。全排序利用递归很容易表达<br/>
